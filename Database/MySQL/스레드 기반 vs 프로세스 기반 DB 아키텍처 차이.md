MySQL과 Oracle의 커넥션 처리 구조 차이

| 항목     | 스레드 기반 (MySQL)           | 프로세스 기반 (Oracle 등)  |
| ------ | ------------------------ | ------------------- |
| 메모리 사용 | 효율적 (공유)                 | 독립적 (많이 씀)          |
| 성능     | 빠름 (전환/통신 빠름)            | 비교적 느림              |
| 안정성    | 낮음 (공유 자원 문제 발생 시 전체 영향) | 높음 (프로세스 간 격리로 안정적) |
| 확장성    | 높음 (가볍게 수천 커넥션 처리 가능)    | 낮음 (많은 접속 시 부담 큼)   |
