ecoli_data 테이블이 존재하고 부모-자식 관계를 나타내는 데이터가 저장되어 있다.
ID 1,2 는 1세대
ID 3,4,5 는 2세대
ID 6,7 은 3세대
ID 8 은 4세대
그럼 ID 를 기준으로 세대를 구하는 쿼리는 어떻게 짤 것인가?
with recursive 문법을 쓰면 쿼리는 다음과 같다.

```
1. 메모리 상에 가상의 테이블을 저장한다.
```
